# 🤖 AI Chat Pro

AI Chat Pro is a modern, interactive chat application built with **Streamlit** that lets you chat with an AI assistant in real-time. It supports multiple chat sessions, code rendering, and tracks response times from start to finish.

---

## 🚀 Features

- **Real-time AI Responses:** Streamed from a local AI model API (Mistral).  
- **User Authentication:** Secure login and registration with hashed passwords using `bcrypt`.  
- **Persistent Chat:** Save, load, and delete chat sessions.  
- **Code Rendering:** Supports syntax-highlighted code blocks in chat.  
- **Dynamic Chat Titles:** Automatically generate short descriptive titles for each chat.  
- **Response Timing:** Shows start time, end time, and duration for each AI response.  
- **Modern UI:** Gradient backgrounds, animated timers, and glowing elements for a sleek look.

---

## 🛠 Tech Stack

- **Frontend:** Streamlit  
- **Backend:** PostgreSQL  
- **AI Integration:** Local AI API (Mistral model via Ollama)  
- **Security:** Password hashing and session management
